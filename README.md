# AluStrap
A 3D printer modelled in OpenSCAD. Most parts are standart off the shelf.

* Frame - square aluminium tubes, 20x20x3mm
* Rods - 12mm diameter
* Bearings - LM12UU and 608ZZ
* Motors - NEMA-17
* Belts and pulleys- GT2
* Threaded rods with nuts - 6mm steel
